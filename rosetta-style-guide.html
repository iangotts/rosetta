How to use this style guide

Copy the entire <style> block from this file into the <head> of any new HTML page. Then paste the logo <img> tag where needed. Use the class names shown below to compose any page layout. Instruct Claude: "Use the Rosetta style guide — nav, hero, sections, footer" and reference this file.

The Google Fonts import at the top of the style block loads DM Serif Display (headlines) and DM Sans (body). Both must be present for the design to render correctly.

Hero Section

Full-width blue gradient with grid overlay and gold radial glow. The inner content is constrained to max-width: 1000px.

<div class="hero"> <div class="hero-inner"> <div class="eyebrow">Eyebrow text here</div> <div class="hero-title">Page Title</div> <div class="hero-subtitle">Supporting subtitle text</div> <div class="hero-meta"> <div class="meta-item">Label <strong>Value</strong></div> <div class="meta-item">Label <strong>Value</strong></div> </div> </div> </div>
Body Sections

Sections alternate white/off-white via :nth-child(even). Always wrap page sections in <div class="page"> (max-width 1000px, centred).

<div class="page"> <section> <div class="sec-label">Section Name</div> <div class="section-headline">Serif headline here</div> <p>Body copy...</p> <div class="horizon"></div> <!-- optional gradient rule --> </section> </div>
