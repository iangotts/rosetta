Identity Mapping

"Show me every object holding customer identity data."

What: Complete Identifier Map

Every field across every object holding identity data (email, phone, name, ID). Includes field API name, type, and population status.

NTO Specific

Surfaces where email lives on Contact vs Account. Critical input for Data 360 matching and deduplication design.

    { "object": "Account", "field": "Phone", "status": "65%", "duplicates": ["Contact.Phone"] }
  ]
}
