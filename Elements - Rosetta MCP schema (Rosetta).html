Question 01
"Show me every object in this org that holds customer identity data..."
What (Identifier Map)
Complete identifier map. Every field across every object that holds identity data. For each: object name, field API name, field type, whether it's populated or empty, and where the same data point exists on multiple objects.
For NTO (Deduplication)
Surfaces where email lives on Contact versus Account versus a custom object, where phone is stored in multiple places, where name fields are duplicated across standard and custom objects. Input to Data 360's matching and deduplication design.
Output Format: JSON Schema 
{
  "reportType": "Identifier Map",
  "organization": "NTO",
  "discoveryDate": "2026-04-30",
  "identityFields": [
    {
      "objectName": "Contact",
      "fieldApiName": "Email",
      "fieldType": "Email",
      "populationStatus": "98%",
      "duplicatesFoundOn": ["Account.Email_Address__c", "VIP_Member__c.Member_Email__p"]
    },
    {
      "objectName": "Account",
      "fieldApiName": "Phone",
      "fieldType": "Phone",
      "populationStatus": "65%",
      "duplicatesFoundOn": ["Contact.Phone"]
    },
    {
      "objectName": "Loyalty_Program__c",
      "fieldApiName": "Loyalty_ID__c",
      "fieldType": "Text(50)",
      "populationStatus": "100%",
      "duplicatesFoundOn": ["Account.Loyalty_ID_Internal__c"]
    }
  ]
}

Question 04
"Which fields on customer objects are populated and which are empty?"
What (Usage)
Field-level usage report. For every customer-related object: which fields are populated, empty, referenced in automation, or referenced in reports. Flags dead weight: fields that are empty and unreferenced.
For NTO (Dead Weight)
Across Account, Contact, Order, Case, and custom objects. Directly answers "which fields should we care about" and removes dead weight before Data 360 design begins.
Output Format: JSON Table Schema 
{
  "usageReport": "Customer-Related Objects",
  "objects": [
    {
      "object": "Account",
      "fieldAnalysis": [
        { "field": "Middle_Name__c", "populated": "2%", "referencedInAutomation": false, "recommendation": "Remove" },
        { "field": "Preferred_Store__c", "populated": "85%", "referencedInAutomation": true, "recommendation": "Keep for Personalization" }
      ]
    }
  ]
}
